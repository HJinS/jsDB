# BUG-001 Root Node 초기 할당 오류

- **커밋:** `a98a1f5`
- **날짜:** 2025-07-03
- **컴포넌트:** `BTree.kt` — `insert`
- **상태:** 수정 완료

## 증상

빈 트리에 첫 번째 키를 삽입할 때 root page가 제대로 할당되지 않아 이후 삽입/검색이 실패.

## 원인

`rootPageId` 초기화 타이밍 오류. 새 페이지를 할당한 뒤 `rootPageId`를 업데이트하기 전에 다른 경로로 분기하는 케이스가 존재.

## 수정

`rootPageId`를 페이지 할당 직후 즉시 갱신하도록 순서 수정.
